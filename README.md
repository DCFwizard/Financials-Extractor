# Financial Statements Extractor  

## Overview  
This Python script allows you to fetch **Balance Sheet, Income Statement, and Cash Flow Statement** for any publicly traded company using `yfinance`. The data is exported to an **Excel file** for easy financial analysis.  

## Features  
✔ Retrieve financial data for the past **four years**  
✔ Automatically sorts and filters financials  
✔ Saves results in an **Excel file**
